TAI LIEU 04 - TANG MODELS (DOMAIN/BUSINESS LAYER)

1) Vi tri
- Thu muc: Source/models
- Thuc the tieu bieu: Account, Admin, Student, Lecturer, Course, ClassSection, Grade, Semester, Faculty...

2) Vai tro
- Dai dien doi tuong nghiep vu va quy tac nghiep vu.
- Chiu trach nhiem cac thao tac domain:
  + Tim theo ID/code,
  + Luu cap nhat,
  + Tinh toan nghiep vu (vi du GPA),
  + Quy trinh nghiep vu dac thu (vi du finalize grades).

3) Vi du quan trong: Admin model
- Admin la trung tam xu ly cac use-case quan tri:
  + createAccount
  + manageStudent / manageLecturer / manageFaculty / manageSemester / manageCourse / manageClassSection
  + assignLecturer
  + finalizeGrades
- finalizeGrades co logic nghiep vu ro:
  + Kiem tra class ton tai,
  + Kiem tra thoi han hoc ky,
  + Chot diem Draft -> Final,
  + Tinh lai GPA sinh vien lien quan.

4) Vi du quan trong: Account model
- Dam nhiem xac thuc (login) va khoi tao tai khoan moi.
- Lien ket chat voi data accounts.txt va auth helper.

5) Nguyen tac phan tang mong muon
- Model nen la noi chua logic nghiep vu cot loi.
- View khong xu ly nghiep vu.
- Utils khong giu state nghiep vu.

6) Van de can theo doi
- Hien tai he thong co dau hieu giao thoa vai tro giua Model va Controller.
- Nen dat quy tac coding convention ro de tranh duplicate.

7) De xuat mo rong
- Them service layer neu nghiep vu tiep tuc tang do phuc tap.
- Them unit test cho cac method domain quan trong (finalize, GPA, assign lecturer...).
