TÀI LIỆU 05 - TẦNG UTILS (INFRASTRUCTURE SUPPORT LAYER)

1) Vị trí
- Thư mục: Source/utils
- File chính:
  + auth_helper.py
  + file_handler.py
  + validators.py

2) auth_helper.py
- hash_password(password): băm SHA-256.
- verify_password(password, hashed): so sánh mật khẩu với hash lưu trữ.
- generate_password(length): sinh mật khẩu ngẫu nhiên đáp ứng độ phức tạp cơ bản.

3) file_handler.py
- DATA_DIR trỏ đến Source/data.
- Các hàm:
  + read_all_lines / write_all_lines / append_line
  + read_records / write_records
  + find_record / find_all_records
  + update_record / delete_record
- Có cơ chế ghi tạm .tmp rồi đổi tên để an toàn dữ liệu hơn khi ghi đè.

4) validators.py
- validate_email
- validate_phone
- validate_password_complexity
- validate_not_empty
- validate_date
- validate_credits
- validate_grade

5) Giá trị của tầng Utils
- Tái sử dụng cao, tránh lặp code.
- Cô lập logic hạ tầng khỏi tầng View/Model.
- Dễ test độc lập từng hàm.

6) Lưu ý kỹ thuật
- SHA-256 là hash nhanh, chưa lý tưởng cho password production.
- Nếu đưa vào môi trường thật, nên dùng bcrypt hoặc argon2.
- validate_date mới kiểm tra cơ bản, chưa xử lý ngày-tháng chi tiết (tháng 2, năm nhuận...) đầy đủ.
