# Student Management System

## Mô tả
Hệ thống quản lý sinh viên

## Công nghệ
- Backend: Python Flask + MySQL
- Frontend: HTML + CSS + JavaScript (Bootstrap 5)

## Chức năng chính
1. Đăng nhập/Đăng xuất
2. Quản lý Sinh viên (CRUD)
3. Quản lý Giảng viên (CRUD)
4. Quản lý Môn học (CRUD)
5. Quản lý Lớp học
6. Quản lý Khoa/Ngành
7. Đăng ký môn học
8. Nhập/Xem điểm

## Cài đặt

### Backend
```bash
cd backend
pip install -r requirements.txt
python run.py
```

### Frontend
Mở file index.html hoặc dùng Live Server
